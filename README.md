# shuttle前端

## 依赖

* JavaScript框架：Vue.js
* UI: Element UI
* 路由：Vue Router
* 全局数据管理：Vuex
* 请求客户端：axios
* 滑动加载：mescroll.js
* FormatTime: moment.js
